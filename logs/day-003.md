# Day 003 — SMB Enumeration and Exploitation Basics

**Source:** TryHackMe

## What I Did
Studied SMB (Server Message Block) — a network protocol used for
file sharing and communication between machines on a network.
Covered enumeration techniques and basic CLI exploitation commands
against misconfigured SMB services.

## What Clicked
SMB is one of the most historically abused protocols in real-world
attacks — EternalBlue, WannaCry, NotPetya all leveraged SMB
vulnerabilities. Understanding why it gets exploited makes the
enumeration process make sense rather than just being a list of
commands to memorize.

Enumeration before exploitation is the mindset shift — you don't
attack blind. You map what's there first.

## What Confused Me
The exact flags and syntax across different tools need more
repetition before they become instinctive. Knowing *what* to
enumerate is clear — the tooling syntax needs more hands-on time.
I messed up the syntax 3 times for not giving a space after smbclient and stating //
