## What I Did
Covered the foundational concepts of Linux — the operating system that
runs practically everything in cybersecurity. Explored the filesystem
structure, basic terminal navigation, and core commands used in
day-to-day security work.

exploit, every script in security runs here. Getting comfortable in
the terminal isn't optional, it's the baseline.
